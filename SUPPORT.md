# Support

If you need help with an issue but do not know where to ask, start a conversation in our [GitHub Discussions](https://github.com/orgs/osafapp/discussions) with the appropriate category.

To report a specific problem or make a detailed request in regards to a design or resource, use GitHub Issues.

To make a comprehensive design contribution, make a GitHub Pull Request.

**For security or privacy related concerns, please see our Security & Privacy Policy.**
